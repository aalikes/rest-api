# Apostille Services

## Overview

Document authentication for international use. An apostille is a certificate that authenticates the origin of a public document, required for documents to be legally recognized in any of the 124 Hague Convention member countries.

## Service Types

### State Apostille
- **Price:** $200 per document
- **Processing:** 3–10 business days (standard), ~5 days (priority)
- **Authority:** Secretary of State in the issuing state
- **Documents:** Birth certificates, marriage certificates, state court documents, notarized documents

### Federal Apostille
- **Price:** $200 per document
- **Processing:** ~5 weeks (standard), ~10 business days (priority)
- **Authority:** U.S. Department of State
- **Documents:** FBI background checks, federal court documents

## Common Use Cases

- **Moving abroad** — residency applications require apostilled documents
- **International marriage** — birth/marriage certificates for foreign marriage
- **Dual citizenship** — birth certificates for citizenship applications
- **Business expansion** — corporate documents for international commerce
- **Education abroad** — diplomas and transcripts for foreign universities
- **International adoption** — multiple documents need apostille
- **Visa applications** — FBI background checks for work/residency visas
